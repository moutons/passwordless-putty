Assumptions

This guide assumes a few things:

  • User is on Windows (Vista or 7) OS (source) and has the ability to install software
  • The machine the user intends to log into (target) is already set up with OpenSSH
  • The user has an account on the target and
    • Is already able to log in from source using an SSH client, OR
    • The target administrator has requested that they provide a public key to set up authentication

There are an ample number of guides to perform various PuTTY related tasks already, but some users I've worked with need a particular set of information that I wasn't seeing. So, writing a screenshot-by-screenshot guide seemed like the thing to do.
